# Airwallex Payments

## Compatibility:
This module was tested on:
* Magento 2.4 @ PHP 7.4

## Installing/Getting started 

```bash
composer require {futer_link}
```

Enable the module
```bash
bin/magento module:enable Airwallex_Payments
```

Module install
```bash
bin/magento setup:upgrade
```

Clean Magento Cache
```bash
bin/magento module:enable cache:flush
```
